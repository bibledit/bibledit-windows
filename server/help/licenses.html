ETCBC4 Hebrew text database

The ETCBC database of the Hebrew Bible contains the scholarly text of the Hebrew Bible with linguistic markup. The source of the data is the HEBREW TEXT DATABASE ETCBC4. Bibledit obtained the data from SHEBANQ, a search engine for the Hebrew Bible. This work is licensed under a Creative Commons Attribution-NonCommercial 4.0 International License.
